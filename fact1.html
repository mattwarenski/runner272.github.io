CSS Layouts

  • CSS has several build in layout tags like < section > and <nav>. these tags can be used to help organize the site and improve accessabilty.
  • Withing the style sheet, the float property can be set to set which side of the screen an object is on.
  • The margin option can be set to increase the white space around the object and the padding option can be used to increase the white space within the object, like this box.
  • Min and max width and heights make the page scalable and make the layout more fluid.
